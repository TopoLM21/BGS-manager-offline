# LeadDangerous

Легкий офлайн-менеджер BGS-планов для Elite Dangerous. Приложение хранит данные
локально в браузере и синхронизирует двух игроков через TXT-отчеты.

## Запуск

Откройте файл `app/index.html` двойным кликом в браузере. Установщик, сервер и
зависимости не нужны.

Данные сохраняются в `localStorage` текущего браузера. Для обмена используйте:

- `Скачать TXT` — выгрузить журнал операций.
- `Загрузить TXT` — применить отчет товарища.

Один и тот же отчет можно импортировать повторно: операции имеют уникальные ID,
поэтому дубли не будут добавлены.

## Что уже есть

- список систем с приоритетом, статусом, целью и заметкой;
- быстрые шаблоны задач;
- поиск системы через EDSM API;
- журнал действий по системам;
- архивирование систем;
- импорт и экспорт TXT-отчетов.

## Формат отчета

Отчет остается обычным текстовым файлом. Значимые строки — JSON-события:

```txt
# LEADDANGEROUS REPORT v1
{"schema":"LeadDangerous.Event/v1","id":"...","type":"system.upsert","payload":{"name":"Sol"}}
{"schema":"LeadDangerous.Event/v1","id":"...","type":"activity.add","payload":{"system":"Sol","text":"Сделал 4 миссии"}}
```

Строки с `#` считаются комментариями.
